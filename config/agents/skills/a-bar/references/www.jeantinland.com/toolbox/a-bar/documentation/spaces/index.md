# Spaces (yabai)

> Published on Wed, Feb 4, 2026> Updated on Wed, Feb 4, 2026

## Description [#](https://www.jeantinland.com/toolbox/a-bar/documentation/spaces/index.html#description)

The Spaces (yabai) widget displays your yabai spaces and the apps running in each space. It can also show sticky windows in a separate group.

This widget requires **yabai** as the active window manager. If you use AeroSpace, see [Spaces (AeroSpace)](https://www.jeantinland.com/toolbox/a-bar/documentation/aerospace-spaces/index.html) instead.

## Interactions [#](https://www.jeantinland.com/toolbox/a-bar/documentation/spaces/index.html#interactions)

* Click a space to focus it.
* Long-press a space to rename it.
* Use the context menu to rename, move, or delete a space.
* Use the **+** button to create a new space (if enabled).

## Available settings [#](https://www.jeantinland.com/toolbox/a-bar/documentation/spaces/index.html#available-settings)

* `Hide empty spaces`: Hide empty, non-visible spaces. **Default is false**.
* `Show all spaces on all screens`: Show every space on every display. **Default is false**.
* `Display sticky windows separately`: Group sticky windows on the left. **Default is true**.
* `Hide duplicate apps`: Hide duplicate app icons within a space. **Default is true**.
* `Excluded spaces`: Comma-separated space labels to hide. **Default is ""**.
* `Title exclusions`: Comma-separated window title patterns to ignore. **Default is ""**.
* `Use regex for exclusions`: Interpret exclusions as regex patterns. **Default is false**.
* `Hide "Add space" button`: Hide the + button. **Default is false**.
* `Switch spaces without yabai`: Use system APIs for space switching. **Default is false**.
