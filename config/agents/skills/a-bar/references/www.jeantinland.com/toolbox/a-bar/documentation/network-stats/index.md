# Network stats

> Published on Wed, Feb 4, 2026> Updated on Wed, Feb 4, 2026

## Description [#](https://www.jeantinland.com/toolbox/a-bar/documentation/network-stats/index.html#description)

The Network stats widget shows upload and download speeds with an overlayed graph.

## Interactions [#](https://www.jeantinland.com/toolbox/a-bar/documentation/network-stats/index.html#interactions)

* Click to open Network Utility (or Activity Monitor on newer macOS versions).

## Available settings [#](https://www.jeantinland.com/toolbox/a-bar/documentation/network-stats/index.html#available-settings)

* `Show icon`: Show or hide the arrows. **Default is true**.
* `Download graph color`: Theme color for download graph. **Default is "magenta"**.
* `Upload graph color`: Theme color for upload graph. **Default is "blue"**.
* `Refresh interval`: Refresh cadence in seconds. **Default is 4**.
