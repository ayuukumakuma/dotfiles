# Process (yabai)

> Published on Wed, Feb 4, 2026> Updated on Wed, Feb 4, 2026

## Description [#](https://www.jeantinland.com/toolbox/a-bar/documentation/process/index.html#description)

The Process (yabai) widget shows the focused application and window title, plus the windows on the current space. It also displays the current layout mode when yabai is active.

This widget requires **yabai** as the active window manager. If you use AeroSpace, see [Process (AeroSpace)](https://www.jeantinland.com/toolbox/a-bar/documentation/aerospace-process/index.html) instead.

## Interactions [#](https://www.jeantinland.com/toolbox/a-bar/documentation/process/index.html#interactions)

* Click a window/app icon to focus it.

## Available settings [#](https://www.jeantinland.com/toolbox/a-bar/documentation/process/index.html#available-settings)

* `Show current space only`: Limit to the focused space. **Default is true**.
* `Hide window title`: Hide the window title line. **Default is false**.
* `Display only icon`: Show icons only. **Default is false**.
* `Show layout type indicator`: Show `bsp`, `stack`, or `float`. **Default is true**.
