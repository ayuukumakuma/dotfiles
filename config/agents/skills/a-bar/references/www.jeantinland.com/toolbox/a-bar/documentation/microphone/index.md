# Microphone

> Published on Wed, Feb 4, 2026> Updated on Sun, Feb 8, 2026

## Description [#](https://www.jeantinland.com/toolbox/a-bar/documentation/microphone/index.html#description)

The Microphone widget shows input level and mute state.

## Interactions [#](https://www.jeantinland.com/toolbox/a-bar/documentation/microphone/index.html#interactions)

* Click to open the microphone volume popover.
* Right‑click to refresh the widget.

## Available settings [#](https://www.jeantinland.com/toolbox/a-bar/documentation/microphone/index.html#available-settings)

* `Show icon`: Show or hide the microphone icon. **Default is true**.
* `Background color`: Theme color for the widget background. **Default is "orange"**.
* `Refresh interval`: Refresh cadence in seconds. **Default is 2**.
