# Appearance settings

> Published on Wed, Feb 4, 2026> Updated on Thu, Feb 12, 2026

## Bar settings [#](https://www.jeantinland.com/toolbox/a-bar/documentation/appearance-settings/index.html#bar-settings)

* `Bar height`: Height in pixels. **Default is 34**.
* `Bar horizontal padding`: Horizontal inner padding in pixels. **Default is 4**.
* `Bar vertical padding`: Vertical inner padding in pixels. **Default is 4**.
* `Bar distance from edges`: Distance from screen edges in pixels. **Default is 0**.
* `Bar radius`: Corner radius in pixels (only applies if distance from edges > 0). **Default is 8**.
* `Bar background opacity`: Opacity percentage (0–100). **Default is 90**.
* `Blur bar background`: Toggle background blur. **Default is true**.
* `Show border`: Toggle bar border. **Default is false**.

**Note:** "Blur bar background" setting will not work if you enabled "Reduce transparency" in macOS Accessibility settings.

## Bar elements settings [#](https://www.jeantinland.com/toolbox/a-bar/documentation/appearance-settings/index.html#bar-elements-settings)

* `Gap between elements`: Spacing between widgets. **Default is 4**.
* `Elements border radius`: Corner radius for elements in pixels. **Default is 4**.

## Data widgets settings [#](https://www.jeantinland.com/toolbox/a-bar/documentation/appearance-settings/index.html#data-widgets-settings)

* `No color in data widgets`: Force monochrome widgets. **Default is false**.

## Font settings [#](https://www.jeantinland.com/toolbox/a-bar/documentation/appearance-settings/index.html#font-settings)

* `Font size`: Global font size in pixels. **Default is 11**.
* `Font`: Optional custom font family.

## Icons settings [#](https://www.jeantinland.com/toolbox/a-bar/documentation/appearance-settings/index.html#icons-settings)

* `Grayscale app icons`: Desaturate app icons. **Default is false**.

**Note:** *a-bar* already handles light/dark/tinted icon variants for system icons and app icons that support it, so you don't need to enable this setting if you just want to match the icons to the theme. This setting is only for users who want to force grayscale icons regardless of theme.

## Appearance [#](https://www.jeantinland.com/toolbox/a-bar/documentation/appearance-settings/index.html#appearance)

* `Appearance`: Auto, Dark, or Light.
* `Dark theme`: Theme preset for dark mode. **Default is Night Shift**.
* `Light theme`: Theme preset for light mode. **Default is Day Shift**.

## Color overrides [#](https://www.jeantinland.com/toolbox/a-bar/documentation/appearance-settings/index.html#color-overrides)

Optional hex color overrides for the current theme (`#RRGGBB`):

* Background
* Foreground
* Main
* Main Alt
* Minor
* Accent
* Red
* Green
* Yellow
* Orange
* Blue
* Magenta
* Cyan
