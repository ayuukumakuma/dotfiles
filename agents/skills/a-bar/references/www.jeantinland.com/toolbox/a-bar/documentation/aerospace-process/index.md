# Process (AeroSpace)

> Published on Mon, Feb 9, 2026> Updated on Mon, Feb 9, 2026

## Description [#](https://www.jeantinland.com/toolbox/a-bar/documentation/aerospace-process/index.html#description)

The Process (AeroSpace) widget shows the focused application and window title for the active AeroSpace workspace. It also displays the other windows on the current workspace as icons.

**Note:** This widget requires **AeroSpace** as the active window manager. If you use yabai, see [Process (yabai)](https://www.jeantinland.com/toolbox/a-bar/documentation/process/index.html) instead.

## Interactions [#](https://www.jeantinland.com/toolbox/a-bar/documentation/aerospace-process/index.html#interactions)

* Click a window/app icon to focus it.

## Available settings [#](https://www.jeantinland.com/toolbox/a-bar/documentation/aerospace-process/index.html#available-settings)

* `Show current space only`: Limit to the focused workspace. **Default is true**.
* `Hide window title`: Hide the window title line. **Default is false**.
* `Display only icon`: Show icons only. **Default is false**.
